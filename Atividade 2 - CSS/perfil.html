Projetos

Projeto Giro Estradas

O Giro Estradas é uma aplicação web desenvolvida com Python e o microframework Flask no backend, projetada especificamente para centralizar a visualização de câmeras de monitoramento de rodovias em uma interface única e ágil. Utilizando HTML, CSS e JavaScript no frontend, o projeto tem como objetivo principal otimizar o fluxo de trabalho em transmissões jornalísticas da Rede Vanguarda, permitindo que a equipe técnica e de jornalismo acesse instantaneamente múltiplos streamings de concessionárias sem a necessidade de navegar por diversos sites externos. Essa arquitetura, integrada via web scraping e consumo de APIs, garante baixa latência e rapidez na captura de imagens em tempo real, servindo como uma ferramenta estratégica para a cobertura de trânsito e segurança viária na região.

Projeto Vancord

O Vancord (Gerenciador de Híbridas) é uma solução de infraestrutura baseada em Python, que utiliza a biblioteca discord.py e o framework PyQt5 para centralizar o controle de bots de voz em uma interface gráfica unificada. O objetivo principal do projeto é permitir que repórteres em trânsito consigam ouvir e se comunicar com a coordenação de links da Rede Vanguarda com alta estabilidade, utilizando a transmissão de áudio local via FFmpeg para conectar microfones e retornos ao Discord. Através de recursos como o agendamento automático de operação e a resiliência garantida pela compatibilidade com a criptografia de ponta-a-ponta (DAVE E2EE), a ferramenta assegura uma comunicação fluida e segura entre as equipes de rua e a central, eliminando falhas críticas de conexão durante as transmissões.
